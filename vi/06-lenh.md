# Lệnh

## Lệnh chính

| Lệnh | Mô tả |
|------|-------|
| `/ego` | Mở menu trạng thái Ego |
| `/ego info` | Xem thông tin tier của bạn |
| `/ego list` | Xem danh sách tier tất cả người chơi |

## Lệnh phụ

| Lệnh | Mô tả |
|------|-------|
| `/sabi vault` | Mở rương riêng |
| `/sabi invvault` | Lưu inventory vào vault |
| `/sabi items` | Lấy item đặc biệt |
| `/sabi mace` | Nhận Mace |
| `/sabi lang <vi/en>` | Đổi ngôn ngữ |

## Lệnh Admin

| Lệnh | Mô tả | Quyền |
|------|-------|-------|
| `/sabi reload` | Tải lại config | `sabismp.admin` |
| `/sabi ego set <player> <tier>` | Đặt tier cho người chơi | `sabismp.admin` |
