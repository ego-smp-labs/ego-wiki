# Mace of Power

> ⚡ Only **ONE** Mace of Power can exist on the entire server at any time!

## Stats

| Property | Value |
|----------|-------|
| Material | Mace |
| Custom Model | `2001` |
| Singleton | ✅ Only one per server |
| Droppable | Configurable (strict mode) |

## Crafting Recipe

```
[ ]  [H]  [ ]
[ ]  [I]  [ ]
[ ]  [B]  [ ]
```

| Slot | Material |
|------|----------|
| **H** | Heavy Core |
| **I** | Netherite Ingot |
| **B** | Breeze Rod |

## ⚠️ Risk: First-Craft Effects

When someone crafts the Mace for the first time:

| Effect | Duration |
|--------|----------|
| **Glowing** | **5 minutes** (300 seconds) |
| **Coordinate Exposure** | All players can see your location |

> Your position is revealed to the ENTIRE server for 5 minutes!

## Strict Mode

When enabled by the admin:
- ❌ Cannot drop the Mace
- ❌ Cannot store in chests, barrels, shulker boxes
- ✅ Can use Anvil and Enchanting Table

## Combat Effects

- **Custom Kill Messages** — unique text when killing with the Mace
- **Ground Slam** — falling blocks effect on impact (configurable, may cause lag)
- **Soul Particles** — optional particle trail when holding
