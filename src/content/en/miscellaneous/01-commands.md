# Commands

## Main Commands

| Command | Description |
|---------|-------------|
| `/ego` | Open Ego status menu |
| `/ego info` | View your tier information |
| `/ego list` | View all players' tier list |

## Secondary Commands

| Command | Description |
|---------|-------------|
| `/sabi vault` | Open personal vault |
| `/sabi invvault` | Save inventory to vault |
| `/sabi items` | Get special items |
| `/sabi mace` | Receive Mace |
| `/sabi lang <vi/en>` | Change language |

## Admin Commands

| Command | Description | Permission |
|---------|-------------|------------|
| `/sabi reload` | Reload config | `sabismp.admin` |
| `/sabi ego set <player> <tier>` | Set player tier | `sabismp.admin` |
